Главный исполняемый файл "MainBaseTone.m", в нем мы запускаем программу. В этой программе мы находим локальные максимумы с помощю окна которое мы сформировали опираять на ширину автокорреляционной функции по половине мошьности сформированой из огибающей спектра полученого из оконного преобразования участка сигнала.
Для старта моделирования в главном файле задайте путь сигнала, размер окна преобразования желательно 512 и выше (кратное степени 2).
